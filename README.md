# 🔍 TruthLens: AI-Powered Credibility Analysis

TruthLens is an advanced AI/NLP-driven platform designed to analyze, classify, and verify information authenticity in real-time. By leveraging state-of-the-art Natural Language Processing techniques, it helps users distinguish between credible content and misinformation.

---

## 🚀 Key Features

* **Advanced NLP Engine:** Uses custom-trained machine learning models to identify patterns, sentiment, and reliability scores.
* **Real-time Analysis:** High-performance API communication between the web interface and the ML backend.
* **Secure Authentication:** Google OAuth integration for seamless user access.
* **Intuitive Dashboard:** A clean, responsive UI built for professional data exploration.
